# CineHub - Movie Discovery Web App (HTML Structure)

CineHub is a movie discovery web application designed to help users explore and browse movies with ease. This HTML structure serves as the foundation for the site, including the layout, sections, and semantic elements.

---

## Project Overview

The HTML defines the structure of the CineHub app, providing a clear layout for featured movies, movie lists, filters, search, watchlist, and modals. It uses semantic tags and IDs to allow dynamic content and styling later.

---

## Features (HTML Structure)

- **Header**
  - Logo
  - Navigation links (Home, Popular, Top Rated, Now Playing, Watchlist)
  - Search bar with input and button

- **Featured Movie Section**
  - Highlights a selected movie at the top
  - Contains image, title, description, and action buttons

- **Movies Header & Filters**
  - Page title
  - Filter section to select genre and year (dropdowns)

- **Movies Grid**
  - Placeholder for dynamically loaded movie cards

- **Empty State**
  - Displayed when no movies are found

- **Watchlist Section**
  - Hidden by default
  - Placeholder for movies added to watchlist

- **Movie Detail Modal**
  - Hidden by default
  - Shows detailed information of a movie including poster, title, overview, release date, rating, and a watchlist button

- **Footer**
  - Project credit

---

## HTML Structure Highlights

- Uses semantic tags: `<header>`, `<main>`, `<section>`, `<footer>`
- Provides IDs and classes for each major section (`moviesGrid`, `watchlistGrid`, `modalBody`, etc.)
- Includes clear inline comments for each section to explain purpose and usage
- Designed to be easily enhanced with styling and scripting

---

## Usage Notes

- This HTML file serves as the skeleton for CineHub.
- Place your images in the `images/` folder (e.g., `inception.jpg`) or replace with placeholders.
- The structure supports future enhancements like dynamic movie loading, filters, search, and modals.
